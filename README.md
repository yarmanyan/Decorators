# Bookkeeping — бухгалтерская программа на Python

## Описание

**Bookkeeping** — это программа для автоматизации бухгалтерского учёта, разработанная на Python. 
Программа позволяет вести учёт сотрудников, расчёт заработной платы.

## Функциональные возможности

- **Учёт сотрудников** — добавление, удаление, обновление данных о сотрудниках
- **Расчёт заработной платы** — автоматический расчёт зарплаты с учётом налогов

## Технологии

- Python 3.9+
- Vitya — валидация российских банковских и налоговых идентификаторов
- Pydantic — валидация данных (версия 1.10)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <url_репозитория>
cd bookkeeping
```

### 2. Установка зависимостей
```bash
python -m venv .venv
```

### 3. Запуск программы
```bash
python main.py
```
